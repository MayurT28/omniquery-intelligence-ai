# 🌐 OmniQuery Intelligence v1.0
website - https://omniquery-intelligence-ai.streamlit.app/

**An AI-Powered Natural Language to SQL Interface**

Developed by **Mayur | MCA Graduate**

## 🚀 Overview
OmniQuery is a Business Intelligence (BI) tool that allows users to query a MySQL database (Sakila) using plain English. It uses **Gemini 3.0 Flash** to interpret intent, generate SQL, and visualize results dynamically.

## 🛠️ Tech Stack
- **Frontend:** Streamlit (Python)
- **AI Engine:** Google Gemini 3.0 API
- **Database:** MySQL / Sakila Schema
- **Visuals:** Plotly Express
- **State Management:** Streamlit Session State

## 🌟 Key Features
- **NLG Summaries:** Custom Python logic for human-like data explanations.
- **Smart Visuals:** Auto-detects data types to show Bar, Pie, or Area charts.
- **Secure Architecture:** Uses `.env` for API security and SQL injection prevention.
